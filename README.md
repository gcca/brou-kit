# Brou-Kit

A Chrome/Brave browser extension that bundles multiple utilities into a single popup.

## Features

- **Zoom** — Control the zoom level of the active tab via slider, number input, or presets (50%–200%). Range: 25%–500%.
- **Tabs** — List all open tabs across windows. Click any tab to switch to it instantly.

Remembers the last selected panel between popup opens.

## Installation

1. Go to `brave://extensions` or `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `brou-kit` folder

## Requirements

- Brave or Chrome 88+
- Manifest V3
